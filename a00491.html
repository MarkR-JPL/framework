Detailed Description

This class maintains the stokes coefficient portion of the state.

This particular implementation uses constant values, combined with fitting for the parallel polarization fraction.

virtual ArrayAd<double, 2> FullPhysics::StokesCoefficientImpBase::stokes_coefficient ( ) const
inlinevirtualinherited

Return Stokes coefficients used to go from Stokes vector to scalar reflectance.

This is number_spectrometer() x 4, and is unit less.

Note that is simple a matter of convenience that we have "4" rather than just number_stokes(). This happens to be how the stokes coefficients are given the Level 1 file. We only actually use the first number_stokes() coefficients.

inlinevirtual

Return a string to identify this part of the state, this name should be all lower case and seperate parts with a /.

For example, an aerosol named strat would be named as: aerosol/strat. A gas named CO2 would be named like this: absorber/co2 The name is intended to be used for looking up retrieval values for a configuration system. Classes that have the same type of inputs should have the same name.
